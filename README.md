# Chat.api Frontend

## Project setup
##### npm
```sh
npm install
```
##### yarn
```sh
yarn install
```
## Installing servers
[Chat.api Backend](https://github.com/Crinax/chat.api-backend-ts-websocket) - Clone this repository and follow installation instruction

## Run app
##### npm
```sh
npm serve
```
##### yarn
```sh
yarn serve
```
